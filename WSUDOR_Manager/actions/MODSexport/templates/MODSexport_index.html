MODS Export/Import

This utility allows you to export multiple MODS records into a <mods:modsCollection> file, or import a collection of MODS records that have been edited outside of Ouroboros (spreadsheet, XML editor, text editors).

NOTE: A WSUDOR convention is to include the PID of an object in the <mods:extension><PID> element. If this is not already present, it will be created on export from this utility. Thus, it is recommended to use the same file you export, for importing.

Choose "export" or "import" below:

  • Export

    For large selections of objects, may take some time. Download will start automatically.
