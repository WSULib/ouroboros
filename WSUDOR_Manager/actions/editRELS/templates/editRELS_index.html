RELS-EXT editing

RDF statements are expressed as triples in Fedora Commons. A triple is comrpised of a "subject", "predicate", and "object". This can be thought of as a "thing (subject)" having a "relationship (predicate" to another "thing (object)". Subject, Predicate, Object is commonly referred to as "SPO". In the case of this utility, the "subject" will always be the current PID in the PID set when iterating through.

Find below methods for adding / editing / removing relationships between objects in Fedora. Please keep in mind, the focus of this utility is to edit objects in bulk

Options:

  • Add Relationships
  • Modify / Purge Shared Relationships
    • As this utility seeks out relationships common to all selected PIDs, it may take some time to load (think 10,000+).
  • Advanced RELS-EXT XML Editing (XML Regex and Blanket String Manipulation)
