Create User

Though Ouroboros uses ldap to authenticate users, it only allows users that have a pre-existing account in the Ouroboros database. By default, a few admin accounts are created. But to open up access to more users, an account must be created for them using the form below.

The form contains minimal information and questions: when the user logs in for the first time, they Display Name and other credentials will be filled in from their LDAP credentials. At this point, just their Access ID is needed which is used to connect with their LDAP authentication.
