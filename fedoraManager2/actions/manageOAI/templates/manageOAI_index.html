Manage OAI Relationships

This utility is designed to manage how objects are exposed via OAI. We currently use PROAI that comes bundled with Fedora. Objects are detected and indexed by PROAI via a small set of RELS-EXT relationships. Find below tools for understanding those relationships, tools for editing them, and general information about the OAI server.

OAI Status and Examples

  • Identify
  • ListIdentifiers
  • Example GetRecord - Letter of appointment from Abraham Lincoln for George Harrington

Generate OAI identifiers

Clicking the button below will create RELS-EXT relationships for all currently selected Objects, that define the unique OAI identifier for that object.

Currently, this only adds a RELS-EXT relationship, meaning multiple are possible if they differ.
